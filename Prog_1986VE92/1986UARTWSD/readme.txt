 Материалы по утилите 1986WSD.exe(программирование флеш-памяти 1986ве9х через UART)
в каталоге 1986WSD представлен проект для MS VisualC++ 2008.
в каталоге 1986bootuart представлен проект для формирования 1986_BOOT_UART.hex в среде Keil 4.0.
файлы 1986_BOOT_UART.hex и 1986WSD.exe должны быть расположены в одном каталоге.
Исполняемый код загрузчика загружается по адресу 0x20007800, его переменные и стек располагаются по адресу 0x20007c00.
Cкорость обмена по RS-232 115200.
После программирования возможен запуск программы из флеш-памяти, нажатием кнопки "RUN".
Имена HEX файла и COM-порта(по умолчанию используется "COM1")задаются перед нажатием кнопки "Start".
Перед нажатием кнопки "Start" МК должен быть введен в режим загрузки через UART(например уставкой соответствующих уровней на входах MODE[2-0]
перед подачей питания на МК).
ПО окончании работы программы имена HEX файла и COM-порта сохраняются в файле 1986WSD.cfg.
