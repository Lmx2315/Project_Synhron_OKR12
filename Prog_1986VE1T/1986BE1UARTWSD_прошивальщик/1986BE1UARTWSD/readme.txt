 Материалы по утилите 1986VE1WSD.exe(программирование флеш-памяти 1986ве1 через UART)
в каталоге 1986VE1WSD представлен проект для MS VisualC++ 2008.
в каталоге 1986ve1bootuart представлен проект для формирования 1986ve1bootuart.hex в среде Keil 3.80.
файлы 1986ve1bootuart.hex и 1986WSD.exe должны быть расположены в одном каталоге.
Режим загрузки через UART - mode[2-0](100b) должен быть задан до подачи питания на МК.
Исполняемый код загрузчика загружается по адресу 0x20103400, его переменные и стек располагаются по адресу 0x20103c00.
Cкорость обмена по RS-232 9600 - 115200.
После программирования возможен запуск программы из флеш-памяти, нажатием кнопки "RUN".
Скорость обмена, имена HEX файла и COM-порта(по умолчанию используется "COM1") задаются перед нажатием кнопки "Start".
ПО окончании работы программы скорость обмена, имена HEX файла и COM-порта сохраняются в файле 1986VE1WSD.cfg.
